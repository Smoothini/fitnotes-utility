FitNotes Utility
This tool is specifically designed for the FitNotes app, which I've happily used for years and can wholeheartedly recommend. It is not affiliated with the app's developer. Always ensure you have a backup of your original .fitnotes file before importing changes!!

Bulk insert new weight values into a FitNotes database

Values only: 1 weight value per line, optionally followed by a comment. Dot for decimals (e.g 75.35)
Manual: 1 set of Date, [Time], Value, [Comment] per line. Separate values with commas, tabs, or spaces.
Time defaults to 09:00:00 if not provided and comment is optional.
Supported date formats: YYYY-MM-DD or DD/MM/YYYY

🔒 Local Processing: All operations happen in your browser's memory. No data is ever uploaded anywhere, and the app also works offline. Feel free to check the source code or verify the network activity to confirm. Again, keep a backup of your original file before making changes!!
